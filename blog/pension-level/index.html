怎樣知道自己的勞保投保薪資級距

2020-03-08

勞保的相關權益皆是以月投保薪資計算，但時常聽到高薪低報的情況，這不僅影響到你的勞工保險理賠金的權益，更會影響到退休金的準備。那麼要如何知道老闆幫自己報的級距是多少呢？此文就是以你的薪資單上勞保自付額來回推你的投保級距。

投保薪資有分級

首先你得知道投保薪資有分等級，這張表常有更動，最新的投保薪資分級表可以到勞動部網站查看(投保薪資分級表傳送門)，108 年的分級為 16 級，最低為 23,100 元/月，最高為 45,800 元/月，你可以用你的薪資單比對一下自己應該是落在那一級，然後再對照每月扣的勞保費是多少。

勞保費部分負擔

廣大的勞工可分為兩種族群，一是受雇者，就是一般的上班族有雇主的，二是自營工作者，自己開店做生意的，這兩種族群的部分分擔不同，如上圖所示，加保在公司，自己負擔比例為 20%，加保在工會，負擔比例則為 60%。

勞保費公式長這樣

勞保費 = 投保薪資 * ( 10% + 1% ) * 負擔比例

負擔比例就看你的身份而定，受雇者用 20%，自雇者用 60% 來算；中間的 10% 就是現行的勞保費率，另外的 1% 則是就業保險的費率。

要來算了喔

情境一：假設你是受雇者，投保級距是 45,800元/月，那麼你的勞保費就會是：

45,800 * ( 0.1 + 0.01 ) * 0.2 = 1,008 (元/月)

情境二：假設你是自行投保工會，投保級距用最低的 23,100 (元/月)，那麼勞保費是：

23,100 * ( 0.1 ) * 0.6 = 1,386 (元/月)

發現了嗎，自雇者少了就保的比例(少了那 1%)，但自己負擔的部分多了 40%，保費會貴上不少，所以許多自雇者投保皆以低級距加保，但勞保給付項目都是以月投保薪資級距來計算，包含生育補助、老年給付、傷病、死亡或失能等，級距不同能領到的錢會差不少喔。

如果你發現薪資單上的勞保費和你算出來的少一點，別覺得是撿到便宜，需要注意自己的薪資是否被老闆少報了，因為那是你自己的權益呀。

不過大家也可以發現，雇主請一個員工要負擔的勞保費用挺高的，而且除了勞保外，還有健保、職災保險、就業保險、勞退等費用，全部累計起來，雇主需要額外負擔每一位員工薪資 * 19.6% 作為費用，假設你的薪資是 40,000 元，那老闆幫你付的保費累計起來是 7,840 元，這樣也不難理解為何會有雇主想要低報薪資了吧，老闆們的營運壓力很大的啊。
